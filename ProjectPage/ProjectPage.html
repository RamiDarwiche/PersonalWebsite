Drone Project

Python | C# | Unreal Engine | SQL

Created in the spring semester of my freshman year for a research course, this project explored using hand gestures for drone flight in construction management. I created a machine learning model using thousands of original images and Google's Teachable Machine to recognize hand gestures for flight controls. The final version of this project was rendered in Unreal Engine using SQLite as a medium between Python and C#.

Digital Twin Project

Python | MQTT | Unreal Engine

A project exploring the applications of machine learning to anomaly detection in construction management. A spreadsheet of over 260,000 mock data points were used to emulate real-time data that served as the basis for the machine learning model. The model was trained to spot anomalies recorded from an HVAC IoT sensor. MQTTX was then used to serve as an information broker from Python to Unreal, where an HVAC system was simulated for testing.
